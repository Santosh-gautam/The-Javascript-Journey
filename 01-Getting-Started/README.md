# Module 01: Getting Started

This module introduces the background architecture of JavaScript. Before writing syntax, we will look at how JavaScript code is parsed, compiled, and executed by modern runtimes.

## 📋 Module Overview

In this module, we will explore:
- What JavaScript is under the hood (dynamic, single-threaded, JIT-compiled).
- How execution runtimes differ (Browser vs Node.js).
- The high-level V8 Engine workflow (Parsing -> AST -> Ignition Interpreter -> TurboFan Optimization).
