# Module 03: JavaScript Fundamentals

This module covers the core building blocks of the JavaScript language. We will explore how memory is allocated for variables, how primitive and reference types differ, and how loops, operators, and conditionals are executed.

## 📋 Module Overview

In this module, we will explore:
- Memory assignment via `var`, `let`, and `const`, including hoisting mechanics and the Temporal Dead Zone (TDZ).
- Data type differences between Stack (primitives) and Heap (reference objects).
- Implicit type coercion rules and strict equality checks.
- Control flow logic optimization and iteration mechanics.
