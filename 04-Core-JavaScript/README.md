# Module 04: Core JavaScript

This module covers the core execution and inheritance mechanics of JavaScript. Understanding these concepts is essential to master scope rules, asynchronous closures, and object-oriented architectures.

## 📋 Module Overview

In this module, we will explore:
- How execution context stack frames are managed.
- Scoping chains and lexical environment boundaries.
- Memory assignment phases (hoisting & TDZ).
- Functional closure bindings.
- Context resolution using the `this` keyword.
- Inheritance using prototype links and ES6 classes.
- Modular setups (ESM vs CommonJS).
- Runtime error propagation and safe handling.
