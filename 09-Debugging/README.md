# Module 09: Debugging

This module covers the systems thinking debugging mindset, stack traces, DevTools workspace bindings, VS Code debugging configurations, breakpoint strategies, async debugging scopes, memory heaps, source maps, and interactive debugging challenges.

## 📋 Module Overview

In this module, we will explore:
- Mental models for debugging scientific hypotheses.
- Parsing error stack frames and runtime exceptions.
- Chrome DevTools features (sources, network, console, rendering).
- VS Code launch profiling configurations.
- Conditional, logpoint, and watch breakpoint strategies.
- Tracing asynchronous promise flows.
- DOM breakpoint mutations and event bindings.
- Heap profiles and allocation diagnostics.
- Production source maps and code mapping.
- Practical Bug Detective challenges.
