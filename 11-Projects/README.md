# Module 11: Projects

This module covers frontend project design specifications, component architectures, mock setups, execution interfaces, and testing strategies for five custom JavaScript applications.

## 📋 Module Overview

In this module, we will explore:
- Specifications for a Todo Application.
- Specifications for a Debounced Search Bar.
- Specifications for a Custom Promise.all runner.
- Specifications for a Custom Event Emitter.
- Specifications for a Mini Virtual DOM engine.
