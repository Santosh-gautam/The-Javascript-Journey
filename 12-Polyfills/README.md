# Module 12: Polyfills

This module covers custom polyfills for core JavaScript array prototype helpers, function execution binders, asynchronous primitives, rate-limiters, object duplicators, and array flattening utilities.

## 📋 Module Overview

In this module, we will explore:
- Array prototype helper polyfills (`map`, `filter`, `reduce`, `forEach`).
- Function execution binder polyfills (`bind`, `call`, `apply`).
- Asynchronous primitive polyfills (`Promise`, `Promise.all`).
- Execution rate-limiter polyfills (`debounce`, `throttle`).
- Complex object utility polyfills (`deepClone`, `flattenArray`).
