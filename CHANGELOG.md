## [1.0.0] - 2026-07-10

### Added

- Created the project repository scaffolding.
- Created root `README.md` introducing the learning pathway, philosophy, and structure.
- Created root `PROGRESS.md` checklist containing v1.0.0 chapters and subsequent module outline mappings.
- Created root `CHANGELOG.md` to track curriculum additions.
- Created module `00-Welcome`, `01-Getting-Started`, `02-Environment-Setup`, and `03-JavaScript-Fundamentals` folder structures.
- Added placeholders for modules `04` to `16` with strict "Scope outline only" warnings.
